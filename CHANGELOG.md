# Changelog

## [1.0.0] - 2026-04-10

### Added

- Initial release of the Terraform Cursor plugin.
- Seventeen skills under `skills/<name>/SKILL.md` covering core language, state, providers, modules, AWS, Azure, GCP, Kubernetes, testing, security, CI/CD, secrets, multi-account, refactoring, OpenTofu, Terragrunt, and troubleshooting.
- Twenty-four command docs under `commands/tf-*.md` with safety guardrails and example invocations.
- `.cursor-plugin/plugin.json` manifest enumerating all skill and command paths.
- `.mcp.json` wiring for the official HashiCorp Terraform MCP server via Docker (`hashicorp/terraform-mcp-server`).
